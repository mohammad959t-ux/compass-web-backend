# Backend

Local API stub for frontend development.

## Setup
1. Copy `.env.example` to `.env` and set:
   - `MONGO_URI`
   - `CLOUDINARY_URL`
   - `ADMIN_EMAIL` / `ADMIN_PASSWORD`
2. Install dependencies from repo root:
   - `npm install`
3. Start the API:
   - `npm run dev:backend`

The API listens on `http://localhost:4000` by default.
